How we intend to operate

Clear before
the customer commits.

These principles guide the final policy. They are intentionally written without fake timelines, blanket promises or supplier assumptions while the OMNI TERRAIN catalog is still being built.

Before a buyer orders 01

Fitment and use case come first.

Technical products can create avoidable returns when the buyer is not shown the important checks. A live listing should help customers confirm their vehicle, boat, RV or system before placing an order.

01
Compatibility

Show the relevant fitment, voltage, connection, size or system information wherever it changes the buying decision.

02
Product details

Use confirmed product data and package contents so buyers do not need to guess what the item includes.

03
Support before checkout

Give the buyer a simple route to ask a question before purchase when the details need confirmation.

Build note: A clear product page reduces avoidable return requests, but never replaces the final published return terms.
Future return workflow 02

Create a fair, documented path.

The final policy will set the exact return eligibility and timing. This preview shows the intended support flow so an investor can see how OMNI TERRAIN plans to handle a genuine customer issue.

01
Request support

The customer provides the order reference, product details and a short explanation of the issue.

02
Review eligibility

Support checks the published terms, product condition, reason for return and any product-specific exception.

03
Provide instructions

When eligible, the buyer receives the correct return or resolution path before sending anything back.

Build note: Return windows, restocking fees, shipping-cost responsibility and any special conditions will be finalized only after supplier agreements are in place.
Damage, wrong item or other issue 03

Resolve the right problem first.

Not every request should follow the same path. A final return policy will distinguish between buyer remorse, incorrect orders, product defects, shipping damage and fulfillment errors.

01
Document the issue

Ask for order details and appropriate photos or evidence when needed to understand the product and delivery condition.

02
Use the correct route

A damaged shipment, wrong item or product question may require a different resolution path from a standard return request.

03
Communicate clearly

The customer should know what information is needed, what happens next and where to ask for help.

Build note: OMNI TERRAIN will not publish a live refund promise until its supplier, inspection, payment and support process can reliably fulfill it.
